# TechPath — Teste Vocacional para Carreira em Tecnologia

O **TechPath** é um projeto web interativo que ajuda pessoas a descobrirem qual área da tecnologia mais combina com seu perfil.

Através de um teste rápido e intuitivo, o usuário recebe um resultado instantâneo com base em suas preferências e habilidades.

---

## Sobre o projeto

Muitas pessoas que querem entrar na área de tecnologia não sabem por onde começar e acabam perdendo tempo estudando conteúdos aleatórios.

O TechPath resolve esse problema oferecendo:

* Direcionamento de carreira
* Resultado rápido (menos de 3 minutos)
* Experiência interativa
* Clareza sobre próximos passos

---

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 📊 Como funciona

1. O usuário acessa o site
2. Responde 15 perguntas
3. O sistema calcula a pontuação
4. Retorna a área mais compatível

### Áreas mapeadas:

* Frontend Development
* Backend Development
* Data Science
* DevOps
* Cybersecurity
* UX Design

## 💰 Modelo de negócio

O projeto foi desenvolvido com foco em monetização:

* Teste 100% gratuito
* Resultado básico liberado
* Conteúdo avançado bloqueado (roadmap, projetos, plano de estudos)
* Venda realizada via plataforma de produtos digitais

---

## 🎯 Objetivo

Criar uma solução simples e escalável para ajudar iniciantes a entrarem na área de tecnologia com mais clareza e direção.
